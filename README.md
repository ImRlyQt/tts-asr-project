# Resume project
This application provides speech-to-text (ASR) and text-to-speech (TTS) services using Whisper for ASR and Coqui TTS for TTS.

### Features
✅ ASR (Whisper) 
✅ TTS (Coqui TTS) 
✅ Flask API 
✅ Streamlit UI 
### How to run
- `python -m venv .venv`
- `pip install -r requirementrs.txt`
- install `mmfpeg`
- `python app.py`
- `streamlit run .\streamlit_app.py`
